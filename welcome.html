Developed and maintained by the Laboratory of Molecular Cytogenetics, Institute of Plant Molecular Biology, Biology Centre AS CR, Ceske Budejovice, Czech Republic.

Galaxy portal is provided within Elixir CZ project. Galaxy server is maintained by CESNET and CERIT-SC that are participants of this project.

RepeatExplorer includes utilities for Graph-based clustering and characterization of repetitive sequences in next-generation sequencing data and tools for the detection of transposable element protein coding domains. Before you can use Galaxy portal, you have to register and your e-mail address have to be verified. This is quite simple process and takes few minutes. All users can use advanced features like data and workflow sharing. Additionally, all registered users are free to use up to 50Gb. Read RepeatExplorer documentation before analyzing data!

REXdb: a reference database of transposable element protein domains

RepeatExplorer2 and DANTE tools utilize reference database of transposable elements (REXdb) for identification and classification of mobile elements. Database is described in the article:  Systematic survey of plant LTR-retrotransposons elucidates phylogenetic relationships of their polyprotein domains and provides a reference for element classification, Mobile DNA 2019, 10:1 https://doi.org/10.1186/s13100-018-0144-1. Standalone database can be downloaded here .
